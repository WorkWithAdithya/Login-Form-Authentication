## Deployment

To deploy this project run

1) Create a folder
2) Open Terminal
3) Run the following commands


```bash
npx create vite@latest
```
4) Name your project
5) Select framework --> React
6) Select Language --> TypeScript
```bash
cd "Your Project Name"
```
```bash
npm install
```
```bash
npm run dev
```
